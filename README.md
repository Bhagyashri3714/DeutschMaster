# DeutschMaster 🇩🇪

DeutschMaster is a German language learning web application built using React and Spring Boot.

## Features

- User Authentication (JWT)
- German Vocabulary
- Grammar Lessons
- Interactive Quizzes
- Flashcards
- A1 and A2 Learning Levels
- Admin Panel for Content Management

## Technologies Used

### Frontend
- React
- Vite
- Bootstrap
- Axios

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- Maven

## Project Structure

```bash
DeutschMaster
│
├── frontend
└── backend
```

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
mvn spring-boot:run
```
